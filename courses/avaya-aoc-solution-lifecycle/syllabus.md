# Avaya AOC — Solution Lifecycle

Lifecycle-focused modules from discovery through retirement, aligned with the **Avaya AOC Solution Lifecycle** library in the application.

---

## How to use this course

1. Work through modules **in order** within each tier unless your instructor directs otherwise.
2. Read each **module guide** (overview, objectives, lessons, takeaways) and complete hands-on steps where suggested.
3. Finish the **quiz** at the bottom of each module file before advancing.
4. Use the **Answer key** only after attempting the questions.
